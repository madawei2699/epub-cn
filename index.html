epub-cn 框架是为了简化中文电子书排版过程所应运而生一套中文 epub 制作框架，包含 epub 基础文件结构以及常的用排版样式，通过 html 标签调用样式，以达到快速、规范排版的目的。V1.1 版本开始增加了第二套模板供选择使用，只需将模板更名为 style.css 即可。所有样式采用 em 单位布局，在 e-ink 设备中更改字体大小，各个部分会按比例缩放，保持协调统一的效果。

为了方便大家快速制作电子书，大部分样式已兼容 epubStar 的导出标准（并对 epubStar 导出的扉页、目录、以及版权页另做了兼容），不兼容的地方通过 “Δ” 在下面的使用范例中标注，这些需要手动修改 html 文件。因 Calibre 导出标准的不规律性，只有标题及正文是兼容的，其他地方需要手动修改。

制作 epub 和使用 epub-cn 框架，都需要有一定的 html 和 css 基础，您可以在 中文 W3school 了解这方面的知识；Markdown 文件编辑器可以快速编辑文本并导出为 html 格式，对制作 epub 有很大帮助，您可以通过 Markdown 语法说明 了解编辑方法；修改 epub，推荐您使用 Sigil，这是一款所见即所得的 epub 制作软件，简单方便。

国外的 e-ink 设备多不支持中文显示，也就是说无法阅读中文书籍，一种办法是 root 后安装中文字体，这种方法解决的比较彻底，但是略微复杂；另一种简单的办法是外挂老牛字体样式*（请查看：http://www.hi-pda.com/forum/redirect.php?goto=findpost&ptid=606293&pid=8088776），阅读时勾选 “publisher default” 调用老牛字体样式中的字体。

epub-cn 框架支持老牛字体样式，所以无论您的设备采用第一种或者第二种解决办法，都可以正常阅读基于 epub-cn 框架制作的中文书籍。

* 老牛字体样式，版权所有：老牛。欢迎非商业性使用，但请注明"老牛字体样式"；商业使用请与老牛联系。

V1.1 版更新（2012年9月30日）

  1. 段落间距由 1.25em 改为 1em；
  2. 注释 <p class="note"> 改为 <div class="notes"> </div> 所包裹的所有 <p> 标签自动作为注释项；
  3. 署名样式 <p class="author"></p> 改为 <p class="signature"></p>
  4. 居左图片或居右图片，宽度从 40% 增长为 47%；
  5. 简单调整了标题及副标题样式，去掉了 h3 和 h4 的第二种样式；
  6. 兼容 epubStar 导出标准，并对 epubStar 导出的扉页、目录、以及版权页另做了兼容；
  7. 修改了 Text 文件夹下的文件的结构，包含了使用范例；
         cover.html —— 封面
         titlepage.html —— 扉页
         contents.html —— 目录
         chapter0.html —— 序
         chapter1.html —— 第一章
         chapter2.html —— 第二章
  8. 做了第二套模板，使用模板只需将模板更名为 style.css 即可。

   特别感谢 k4user 和 larryli 两位前辈对新版本制作所给予的支持和帮助！

• 如果源文件已经在段首空两个字符表示首行缩进，可以直接删除 template1.css 第 140 行，或 template2.css 第 157 行的 “text-indent: 2em;”

图文混排

<img src="" alt=""/>

<img class="left" src="" alt=""/>

<img class="right" src="" alt=""/>

• 默认居中显示，90% 屏幕宽度，添加相对应的 class 可居左或居右，47% 屏幕宽度

• 如果将图片空放在 <body> 下，或者 <p> 的开头，会因为默认首行缩进造成偏移，因此最好将图片插入到文本中间

（注：偏移的问题只在用电脑查看时出现，设备中还是正常显示的）

• 给注释添加 id="note1"，配合 <a href="#anchor1"></a> 实现跳转到注释
